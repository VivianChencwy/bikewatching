An interactive geospatial visualization of Bluebikes traffic patterns in the Boston and Cambridge area.

## Features

- **Interactive Map**: Pan and zoom around the Boston/Cambridge area
- **Bike Lanes**: Visual display of bike lanes from Boston and Cambridge datasets
- **Station Traffic**: Circle markers showing traffic volume at each Bluebikes station
- **Traffic Flow**: Color-coded circles indicating departure vs. arrival patterns
- **Time Filtering**: Interactive slider to filter data by time of day
- **Tooltips**: Hover over stations to see detailed traffic statistics

## Tech Stack

- **Mapbox GL JS**: For the interactive base map
- **D3.js**: For data processing and SVG overlays
- **Bluebikes GBFS API**: Real-time station information
- **GeoJSON**: Boston and Cambridge bike lane datasets

## Data Sources

- Bluebikes trip data (March 2024): ~260,000 rides
- Boston bike lanes: Open Data Boston
- Cambridge bike lanes: Open Data Cambridge
- Station information: Bluebikes GBFS feed
